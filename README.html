Installation - Windows

To install the Space Invaders on Windows platform, you will require Python programming language.

The game was tested under Python 3.5 / 3.6 / 3.7; however, the game may work under other versions as well.

Python programming language can be downloaded from https://www.python.org

The game should be copied to desired folder, and ran using 'main.py' file

Installation - Linux

Disclaimer: The game have not been tested yet under linux platform.

The game requires Python3 on linux. On new distributions this comes preinstalled

If this is not the case download python deb package and install it from https://www.python.org

Once the python is installed, you can either run the game by doing the following

  • Open terminal
  • Navigate to the location where game was extracted
  • Execute 'chmod +x main.py'
  • Execute './main.py'
or by
  • Open terminal
  • Navigate to the location where game was extracted
  • Execute 'python3 main.py'

The game in current state does not support sound on Linux platform
